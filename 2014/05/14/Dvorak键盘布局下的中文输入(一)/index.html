Dvorakй”®зӣҳеёғеұҖдёӢзҡ„дёӯж–Үиҫ“е…Ҙ(дёҖ)

еҺҹж–ҮпјҡDvorakй”®зӣҳеёғеұҖдёӢзҡ„дёӯж–Үиҫ“е…Ҙ(дёҖ)

еҰӮжһңжӮЁиҝҳжІЎжңүеҗ¬иҜҙиҝҮDvorak(еҫ·жІғеӨҸе…Ӣ)й”®зӣҳеёғеұҖпјҢеҸҜд»Ҙе…Ҳд»Һз»ҙеҹәзҷҫз§‘дёҠдәҶи§ЈдёҖдёӢзӣёе…іиғҢжҷҜпјҡ
http://zh.wikipedia.org/wiki/еҫ·жІғеӨҸе…ӢйҚөзӣӨ

жҲ‘жҳҜдёҖдёӘйҮҚеәҰзҡ„DvorakдҪҝз”ЁиҖ…пјҢеҖҳиӢҘз»ҷжҲ‘дёҖеүҜжҷ®йҖҡзҡ„Qwertyй”®зӣҳпјҢжҲ‘еҸӘиғҪзӣҜзқҖе®ғпјҢз”ЁдёӨдёӘжүӢжҢҮвҖңжҲіеӯ—вҖқгҖӮжҲ‘зҡ„зӣёеҶҢдёӯжңүдёҖеј еӣҫзүҮпјҢжҳҜжҲ‘жӯЈеңЁдҪҝз”Ёзҡ„MacbookпјҢжҢүй”®е·Із»Ҹиў«жҲ‘йҮҚж–°жҺ’дәҶеә§ж¬ЎгҖӮ

GoogleдәҶдёҖйҖҡпјҢдјјд№Һзӣ®еүҚжңҖеҝ«жңҖжөҒиЎҢзҡ„йҖҹи®°ж–№жі•з§°дёәвҖңдәҡдјҹйҖҹеҪ•вҖқпјҢдҪҝз”Ёдё“й—Ёе®ҡеҲ¶зҡ„й”®зӣҳе’Ңиҫ“е…Ҙжі•гҖӮе°Ҫз®Ўж— жі•еңЁз”өи„‘дёҠжЁЎжӢҹпјҢдҪҶжҳҜе…¶ж ёеҝғеҶ…е®№еӨ§жҰӮжңүдёүзӮ№еҸҜд»ҘеҗёеҸ–пјҡ

  1. ж №жҚ®дёӯж–Үд№ жғҜйҮҚж–°е®үжҺ’дәҶй”®дҪҚпјҢеёёз”ЁжҢүй”®еңЁдёӯй—ҙдёҖжҺ’гҖӮ
  2. е°ҶеӨҡдёӘеӯ—з¬Ұз»„жҲҗзҡ„йҹөжҜҚе®үжҺ’еңЁдәҶеӣәе®ҡзҡ„жҢүй”®дёҠпјҢзұ»дјјвҖңеҸҢжӢјвҖқпјҢеҸӘз”Ёиҫ“е…ҘдёҖдёӘеӯ—з¬ҰгҖӮ
  3. еҰӮеҗҢдҪҝз”Ёд№җеҷЁдёҖиҲ¬пјҢеӨҡдёӘжҢүй”®еҸҜд»ҘеҗҢж—¶жҢүдёӢгҖӮ

Dvorakй”®зӣҳеёғеұҖдёӢдёӯж–Үиҫ“е…Ҙжі•зҡ„и®Ёи®әеҮ д№ҺжҳҜдёӘз©әзҷҪпјҢжҲ‘жғіеңЁиҝҷдёҖзі»еҲ—ж–Үз« дёӯеҜ№дёҠйқўдёүзӮ№йҖҗдёҖеҲҶжһҗпјҢзңӢзңӢжҳҜеҗҰиғҪеӨҹжүҫеҲ°дёҖдәӣжҸҗй«ҳеҪ•е…ҘйҖҹеәҰзҡ„еҠһжі•гҖӮд»ҠеӨ©жҳҜз¬¬дёҖйғЁеҲҶпјҢеҜ№DvorakеңЁдёӯж–Үиҫ“е…Ҙзҡ„й”®дҪҚе®үжҺ’ж–№йқўеҒҡдёҖзӮ№з ”з©¶гҖӮ

еңЁDvorakй”®зӣҳеёғеұҖдёӢпјҢиӢұж–ҮжңҖеёёз”Ёзҡ„еӯ—жҜҚиў«е®үжҺ’еҲ°дәҶжүӢжҢҮжңҖе®№жҳ“жҢүеҲ°зҡ„ең°ж–№гҖӮеҜ№иҝҷдёҖзӮ№зҡ„з ”з©¶е·Із»ҸеҫҲеӨҡпјҢдёҚз”ЁиҙЁз–‘гҖӮеҸҜжҳҜз”ЁжқҘиҫ“е…Ҙдёӯж–ҮпјҲжӢјйҹіпјүзҡ„жғ…еҶөеҸҲжҳҜжҖҺд№Ҳж ·е‘ўпјҹ

д»ҺеүҚжңүдәәеҜ№жұүеӯ—еӯ—з¬ҰйӣҶеҒҡиҝҮз»ҹи®ЎпјҢеңЁDvorakеёғеұҖдёӢпјҢй”®зӣҳзҡ„дёӯжҺ’дҪҝз”ЁзҺҮиҰҒжҳҺжҳҫй«ҳдәҺQwertyпјҢиҝҷд№ҹе°ұж„Ҹе‘ізқҖеҗҢиҫ“е…ҘиӢұж–ҮдёҖж ·пјҢиҫ“е…ҘжұүиҜӯжӢјйҹід№ҹиғҪд»ҺиҜҘеёғеұҖдёӯеҸ—зӣҠпјҢжүӢжҢҮз§»еҠЁзҡ„и·қзҰ»жӣҙе°‘гҖӮдҪҶжҳҜеҜ№еӯ—з¬ҰйӣҶзҡ„з»ҹи®ЎжҜ•з«ҹд»ЈжӣҝдёҚдәҶе®һйҷ…жғ…еҶөпјҢеӣ дёәеңЁе®һйҷ…иҫ“е…ҘдёӯпјҢеҗ„дёӘжұүеӯ—зҡ„дҪҝз”Ёйў‘зҺҮдёҚеҗҢпјҢеҸҜиғҪйҖ жҲҗе®һйҷ…зҡ„дҪҝз”Ёз»“жһңдёҺеҜ№еӯ—з¬ҰйӣҶзҡ„з»ҹи®ЎеӯҳеңЁе·®еҲ«гҖӮжүҖд»ҘDvorakзҡ„ж•ҲзҺҮз©¶з«ҹеҰӮдҪ•пјҢиҝҳиҰҒеңЁе®һи·өдёӯжЈҖйӘҢгҖӮ

еңЁдёӢйқўиҝҷж®өPythonд»Јз ҒдёӯпјҢжҲ‘еҜ№иҮӘе·ұд»ҘеүҚж’°еҶҷзҡ„еҮ зҜҮзЁҝд»¶иҝӣиЎҢдәҶеҲҶжһҗгҖӮиҝҷдәӣзЁҝд»¶еҶ…е®№еӨ§еӨҡдёҺITзӣёе…іпјҢеӯ—ж•°зәҰдёәдә”гҖҒе…ӯдёҮгҖӮ

#!/usr/bin/env python
# -*- coding: utf-8 -*-

__author__ = "ldx (lispython@gmail.com)"
__date__   = "Sat Sep 26 08:02:00 2009"


#from pinyin import hanzi2pinyin
from hanzimap import hanzi2pinyin
import collections
import numpy as np
import matplotlib.pyplot as plt

doc = open('doc.txt', 'r').read()  
rst = hanzi2pinyin(doc.decode('utf8'))
result = {}
# print rst.count('iong')

data = collections.defaultdict(int)

all_keys = '\',.;abcdefghijklmnopqrstuvwxyz'
dv_rows = ['\',.pyfgcrl', 'aoeuidhtns', 'qjkxbmwvz']
qw_rows = ['qwertyuiop', 'asdfghjkl', 'zxcvbnm,.']

for char in rst:
    if char in all_keys:
        data[char] += 1
        data['letters'] += 1


for c in all_keys:
    result[c]=(float(data[c])/data['letters']*100)


dv = [0,0,0]
qw = [0,0,0]

for i in range(3):
    for c in dv_rows[i]:
        dv[i] += result[c]

for i in range(3):
    for c in qw_rows[i]:
        qw[i] += result[c]


keys = result.keys()
keys.sort()
val = [result[key] for key in keys]

plt.figure(1, figsize=(8,8))
ax = plt.axes([0.1, 0.1, 0.8, 0.8])

labels = ('up', 'middel', 'down')

explode = (0.05, 0.05, 0.05)
plt.pie(dv, explode=explode, labels=labels, autopct='%1.1f%%', shadow=True)
plt.title('Dvorak', bbox={'facecolor':'0.8', 'pad':5})

plt.figure(2, figsize=(8,8))
plt.pie(qw, explode=explode, labels=labels, autopct='%1.1f%%', shadow=True)
plt.title('Qwerty', bbox={'facecolor':'0.8', 'pad':5})

pos = np.arange(0, 60, 2)
plt.figure(3)
plt.barh(pos, val, 1.5, align='center', alpha = 0.8)
plt.yticks(pos, keys)
plt.xlabel('Percentage')
plt.grid(True)

plt.show()

еҜ№дәҺиҝҷж®өд»Јз ҒжңүеҮ зӮ№йңҖиҰҒиҜҙжҳҺпјҡ

  1. е°Ҷжұүеӯ—иҪ¬жҚўдёәжӢјйҹідҪҝз”ЁдәҶpyzhдёӯзҡ„pinyinжЁЎеқ—пјҢдҪҶжҳҜиҜҘжЁЎеқ—дёҚиғҪе°Ҷдёӯж–Үж ҮзӮ№иҪ¬жҚўдёәиӢұж–Үж ҮзӮ№иҝӣиЎҢз»ҹи®ЎпјҢеӣ жӯӨжҲ‘еҒҡдәҶдёҖзӮ№дҝ®ж”№пјҢе°Ҷе…¶е‘ҪеҗҚдёәhanzimapгҖӮ
  2. д»Јз ҒдёӯдҪҝз”ЁдәҶmatplotlibеә“пјҲеҗҢж ·д№ҹдёҚеңЁж ҮеҮҶеә“дёӯпјүпјҢе°Ҷз»ҹи®Ўз»“жһңз”ҹжҲҗеӣҫеғҸгҖӮ
  3. дёӯж–Үж–ҮзЁҝж ·жң¬дёәdoc.txtпјҢдҪҝз”Ёutf-8зј–з ҒпјҢдёҺд»Јз Ғж”ҫеңЁеҗҢдёҖзӣ®еҪ•дёӢгҖӮж–ҮзЁҝдёӯзҡ„з©әзҷҪдёҚдјҡеҪұе“ҚжңҖеҗҺз»“жһңгҖӮ

з»“жһңеҰӮдёӢпјҡ

еҲҶеҲ«еңЁQwertyе’ҢDvorakеёғеұҖдёӢпјҢй”®зӣҳзҡ„дёҠгҖҒдёӯгҖҒдёӢдёүжҺ’дҪҝз”Ёжғ…еҶөеҜ№жҜ”гҖӮ

жҲ‘д»¬еҸҜд»ҘзңӢеҲ°пјҲжё…жҷ°дёҖдәӣзҡ„еӣҫзүҮеҸҜд»ҘеңЁжҲ‘зҡ„зӣёеҶҢдёӯжөҸи§ҲпјүQwertyеёғеұҖдёӯй—ҙдёҖжҺ’зҡ„дҪҝз”ЁзҺҮдёә34.8пј…пјҢз”ҡиҮіиҝҳдёҚеҰӮдёҠйқўдёҖжҺ’гҖӮ

DvorakеёғеұҖдёӢпјҢдёӯжҺ’зҡ„дҪҝз”ЁзҺҮеҲҷиҫҫеҲ°дәҶ69пј…пјҢеҮ д№ҺжҳҜQwertyеёғеұҖзҡ„дёӨеҖҚгҖӮзЁҚеҫ®жңүзӮ№е„ҝеҸҜжғңзҡ„жҳҜеӨ„дәҺдёҠйқўдёҖжҺ’зҡ„еӯ—жҜҚgпјҢе®ғзҡ„дҪҝз”ЁзҺҮеҫҲй«ҳпјҢз”ҡиҮій«ҳиҝҮдәҶдёӯжҺ’зҡ„и®ёеӨҡеӯ—жҜҚпјҢдёҚиҝҮеҘҪеңЁиҜҘеӯ—з¬ҰдҪҚдәҺеҸіжүӢйЈҹжҢҮдёҠж–№пјҢжҳҜйҷӨдёӯжҺ’д»ҘеӨ–жңҖе®№жҳ“и§ҰеҲ°зҡ„жҢүй”®гҖӮ
