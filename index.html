еҶӣи®ӯж—¶еҸ—еҲ°дёҘиӢӣзҡ„и®ӯз»ғпјҢз»“жқҹж—¶жүҖжңүдәәеҚҙжҠұзқҖж•ҷе®ҳз—ӣе“ӯжөҒж¶•пјҹ

еҰӮжһңи§үеҫ—вҖңе…ідҪ еұҒдәӢвҖқвҖңе…іжҲ‘еұҒдәӢвҖқзҡ„йҖјж јдёҚеӨҹй«ҳзҡ„иҜқпјҢе°ұз”ЁвҖңзұіе…°жҳҶеҫ·жӢүвҖқеҗ§гҖӮ

еҺҹж–Үпјҡhttp://www.zhihu.com/question/19981962

иҝҷдёҚжҳҜж–Ҝеҫ·е“Ҙе°”ж‘©з»јеҗҲз—ҮпјҢиҖҢжҳҜзұіе…°жҳҶеҫ·жӢүжҸҸиҝ°зҡ„вҖңеӘҡдҝ—вҖқгҖӮ

зұіе…°жҳҶеҫ·жӢүзҡ„вҖңеӘҡдҝ—вҖқжҳҜдёҖдёӘжҜ”иҫғйҡҫд»ҘзҗҶи§Јзҡ„жҰӮеҝөгҖӮд»–дёҫдҫӢиҜҙпјҢиҚүеқӘдёҠжңүдёҖзҫӨеӯ©еӯҗеңЁеӨ§з¬‘зқҖеҘ”и·‘пјҢдәәд»¬жӯЈеёёзҡ„еҸҚжҳ еҪ“з„¶жҳҜи§үеҫ—ж„ҹеҠЁпјҢи§үеҫ—жё©йҰЁзӯүзӯүгҖӮдҪҶдёҖдёӘдәәеҸҜдёҚеҸҜд»ҘйқўеҜ№иҝҷж ·зҡ„еңәйқўж— еҠЁдәҺиЎ·пјҢжҲ–иҖ…ж„ҹеҲ°еҺҢжҒ¶пјҹзұіе…°жҳҶеҫ·жӢүи®ӨдёәпјҢеҪ“з„¶жҳҜеҸҜд»Ҙзҡ„гҖӮдҪҶеңЁзҺ°е®һз”ҹжҙ»дёӯпјҢеӨ§е®¶дјҡи§үеҫ—пјҢйқўеҜ№иҝҷж ·зҡ„еңәйқўж— еҠЁдәҺиЎ·зҡ„дәәжҳҜеҶ·иЎҖзҡ„пјҢиҮіе°‘жҳҜдёҚжӯЈеёёзҡ„пјҢжҜҸдёӘдәәйғҪжӢ…еҝғиҮӘе·ұиў«зңӢжҲҗйӮЈдёӘдёҚжӯЈеёёзҡ„дәәпјҢдәҺжҳҜпјҢзңӢеҲ°е°Ҹеӯ©е’ҢиҚүеқӘзҡ„еңәжҷҜе°ұдјҡиЎЁзҺ°еҮәж„ҹеҠЁгҖҒжё©йҰЁзҡ„еҸҚеә”пјҢд»ҘжұӮеҫ—йӮЈз§ҚиһҚе…Ҙдәәзұ»йӣҶдҪ“зҡ„е®үе…Ёж„ҹгҖӮд№…иҖҢд№…д№ӢпјҢиҝҷз§ҚеҸҚжҳ жҲҗдәҶдёҖз§ҚдёҚз”Ёз»ҸиҝҮеӨ§и„‘жҖқиҖғзҡ„жқЎд»¶еҸҚе°„пјҢеҸҚиҖҢйҒ®и”ҪдәҶжҲ‘д»¬жӯЈеёёзҡ„еҝғзҗҶж„ҹеҸ—гҖӮ

иҝҷж ·зҡ„еңәжҷҜеңЁз”ҹжҙ»дёӯжңүеҫҲеӨҡгҖӮдәІдәәеҺ»дё–пјҢдҪ еә”еҪ“жӮІдјӨпјҢжңӢеҸӢзҰ»еҲ«пјҢд№ҹеә”еҪ“жӮІдјӨпјҢжҒӢдәәеҮәиҪЁпјҢдҪ еә”еҪ“ж„ӨжҖ’пјҢиҝҷз§Қжғ…ж„ҹе’ҢеҜ№еә”зҡ„еңәжҷҜпјҢж—©е°ұйҖҡиҝҮеҗ„з§Қж–№ејҸпјҢеӣәеҢ–еңЁжҲ‘д»¬и„‘жө·дёӯпјҢз”ҡиҮіеңЁеҫҲеӨҡжғ…еҶөдёӢпјҢйҒ®и”ҪдәҶжҲ‘д»¬зҡ„зңҹе®һж„ҹеҸ—гҖӮ

дёәд»Җд№ҲйқўеҜ№еҶӣи®ӯз»“жқҹиҝҷж ·зҡ„еңәжҷҜпјҢжөҒжіӘжҳҜеҫҲеӨҡдәәзҡ„еҸҚеә”ж–№ејҸпјҹ

еҜ№еӨ§еӨҡж•°дәәжқҘиҜҙпјҢеҶӣи®ӯе°ұжҳҜеҗғиӢҰпјҢдҪҶиҝҷз§ҚеҗғиӢҰжҳҜжІЎжңүж„Ҹд№үзҡ„пјҢжҳҜиў«ж Ўж–№ејәеҲ¶еҸӮдёҺзҡ„гҖӮдҪҶдәәд»¬еҜ№иҝҷз§Қд»·еҖјзҡ„иҷҡж— жҳҜжҒҗжғ§зҡ„пјҢд»–д»¬дёҚж„ҝж„Ҹжүҝи®ӨиҮӘе·ұжҳҜзҷҪзҷҪеҸ—иӢҰпјҢйӮЈж ·зҡ„жү“еҮ»еӨӘеӨ§дәҶпјҢе№¶дё”иҝҷе®№жҳ“жҝҖиө·дәәд»¬еҸҚжҠ—зҡ„ж¬ІжңӣпјҢдҪҶеҸҚжҠ—еҸҜиғҪж„Ҹе‘ізқҖжӣҙеӨ§зҡ„жү“еҮ»гҖӮ

дёәдәҶж¶Ҳи§Јиҝҷз§ҚжҒҗжғ§пјҢдәәд»¬еӣһжңӣиҮӘе·ұз»ҸеҺҶзҡ„жІЎжңүд»·еҖјзҡ„иӢҰйҡҫзҡ„ж—¶еҖҷпјҢжҖ»жҳҜиҜ•еӣҫз”ЁдёҖз§Қж„ҹеҠЁзҡ„ж–№ејҸпјҢжқҘдёәиҝҷж®өиӢҰйҡҫиөӢдәҲдёҖз§ҚеҙҮй«ҳж„ҹпјҢеҖҹжӯӨжқҘе®үж…°иҮӘжҲ‘пјҢйҖғйҒҝд»·еҖјзҡ„иҷҡж— пјҢж¶Ҳи§ЈиҮӘе·ұеҸҚжҠ—зҡ„ж¬ІжңӣгҖӮ

дҫӢеҰӮпјҢеҫҲеӨҡзҹҘйқ’зҡ„йқ’жҳҘеҺҶз»ҸдәҶжң¬дёҚеә”иҜҘз»ҸеҺҶзҡ„зЈЁйҡҫпјҢдҪҶд»–д»¬дёӯзҡ„еҫҲеӨҡдәәиҮід»Ҡд»Қе®Јз§°вҖ”вҖ”вҖңйқ’жҳҘж— жӮ”вҖқгҖӮеҗҢж ·еҫҲеӨҡжңүиҝҮеҶӣж—…з”ҹж¶Ҝзҡ„дәәд№ҹз»ҸеёёиҜҙвҖңеҪ“е…өеҗҺжӮ”дёүе№ҙпјҢдёҚеҪ“е…өеҗҺжӮ”дёҖиҫҲеӯҗвҖқпјҢдҪ и§ҒиҝҮиҜ»еӨ§еӯҰеҮәжқҘзҡ„дәәиҜҙиҝҮзұ»дјјзҡ„иҜқд№ҲпјҹеҪ“дәәд»¬жңүеҝ…иҰҒе®Јз§°жҹҗж®өз»ҸеҺҶвҖңж— жӮ”вҖқзҡ„ж—¶еҖҷпјҢиҮіе°‘иҜҙжҳҺиҝҷж®өз»ҸеҺҶжңүи®©дәәеҗҺжӮ”зҡ„зҗҶз”ұгҖӮ

иҝҷз§ҚйқўеҜ№иӢҰйҡҫзҡ„ж–№ејҸпјҢжҳҜдёҖз§Қз§ҜжһҒзҡ„еҝғжҖҒпјҢжң¬ж— еҸҜеҺҡйқһгҖӮдҪҶжҳҜпјҢз”ұдәҺжҹҗдәӣеҺҹеӣ пјҢиҝҷз§ҚдёҖеӣһжңӣиӢҰйҡҫе°ұејҖе§ӢиҮӘжҲ‘ж„ҹеҠЁзҡ„еңәйқўпјҢжёҗжёҗзҡ„з¬ҰеҸ·еҢ–дәҶгҖҒжЁЎејҸеҢ–дәҶгҖӮдәәд»¬йҒҮи§Ғиҝҷж ·зҡ„еңәйқўпјҢе·Із»ҸдёҚз”ЁжҖқиҖғпјҢзӣҙжҺҘд»ҺвҖңж•°жҚ®еә“вҖқдёӯи°ғз”ЁдҫҝжҳҜгҖӮе№¶дё”иҝҷз§Қжғ…ж„ҹеҸҚеә”пјҢеҰӮеҗҢжҙӘж°ҙдёҖиҲ¬пјҢе®Ңе…Ёж·№жІЎдәҶйӮЈдәӣеҜ№иӢҰйҡҫиЎЁзӨәжҶҺжҒЁпјҢиЎЁзӨәж„ӨжҖ’гҖҒз”ҡиҮіиЎЁзӨәеҸҚжҠ—зҡ„йӮЈдёҖйғЁеҲҶдәәзҡ„жғ…ж„ҹгҖӮ

еӨ§е®¶еӣһжғідёҖдёӢпјҢеҪ“еҶӣи®ӯз»“жқҹпјҢеӨ§е®¶йғҪеңЁз”ЁзңјжіӘдёәиҝҮеҺ»зҡ„иҝҷж®өж—¶е…үиөӢдәҲж„Ҹд№үпјҢдҪ дёҚеҸӮдёҺпјҢдҪ е°ұжҳҜејӮзұ»гҖӮеӨ§е®¶йғҪеңЁдёәйӣҶдҪ“зҡ„и§Јж•Јж„ҹеҲ°жӮІдјӨпјҢдҪ дёҚжӮІдјӨпјҢдҪ е°ұжҳҜеҶ·жј гҖӮеңЁиҝҷз§Қжғ…еҶөдёӢпјҢдҪ дёҚжөҒжіӘпјҢжҳҜдёҚжҳҜжңүдёҖз§Қиў«йӣҶдҪ“жҠӣејғзҡ„жҒҗжғ§ж„ҹпјҹиҖҢдҪ еҸӮдёҺдәҶпјҢе°ұиҺ·еҫ—дәҶдёҖз§ҚиһҚе…ҘйӣҶдҪ“зҡ„е®үе…Ёж„ҹгҖӮеҪ“дёҖдёӘдәәйқўеҜ№иҝҷж ·зҡ„еңәйқўпјҢдёҚз»ҸиҝҮиҮӘиә«зҡ„жҖқиҖғпјҢиҖҢи®©иҮӘе·ұйҡҸзқҖзҫӨдҪ“зҡ„ж„ҹжғ…зҡ„жҙӘжөҒиҖҢеҺ»пјҢиҝҷе°ұжҳҜеӘҡдҝ—гҖӮ

жүҖд»ҘпјҢеӯ©еӯҗпјҢж·Ўе®ҡпјҢдҪ е’Ңж•ҷе®ҳд»Қз„¶дёҚиҝҮжҳҜиҗҚж°ҙзӣёйҖўиҖҢе·ІгҖӮеӨ§е®¶йғҪйҖўеңәдҪңжҲҸпјҢз»“жһңйғҪе…ҘжҲҸдәҶпјҢе°ұиҝҷд№ҲеӣһдәӢгҖӮ

е°Өе…¶еҖјеҫ—жіЁж„Ҹзҡ„жҳҜпјҢиҝҷз§ҚвҖңеӘҡдҝ—вҖқеңЁеҶӣиҗҘж–ҮеҢ–йҮҢжңҖдёәзӣӣиЎҢгҖӮеӣ дёәеҶӣйҳҹзү№еҲ«йңҖиҰҒеЈ«е…өеӣўз»“дёҖиҮҙпјҢе№¶дё”еңЁеҝ…иҰҒж—¶зҢ®еҮәз”ҹе‘ҪпјҢеӣ жӯӨеёёеёёйңҖиҰҒеңЁеҶӣйҳҹиҗҘйҖ еҮәдёҖз§ҚеҘүзҢ®з”ҹе‘Ҫзҡ„еҙҮй«ҳж„ҹгҖҒеЈ«е…өд№Ӣй—ҙйӮЈз§Қз”ҹжӯ»дёҺе…ұзҡ„жҲҳеҸӢжғ…гҖӮиҖҢиҝҗз”ЁвҖңеӘҡдҝ—вҖқе°ұжҳҜжңҖеёёз”Ёзҡ„е®Јдј ж–№ејҸпјҢжҜ”еҰӮе®ҳж–№з»„з»ҮиӘ“еёҲеӨ§дјҡгҖҒйҖҒйҖҖдјҚжҲҳеҸӢзӯүзӯүйҮҚеӨ§еңәйқўпјҢйғҪжңүеҠ©дәҺе”Өиө·е®ҳж–№йңҖиҰҒзҡ„жғ…ж„ҹгҖӮжүҖд»ҘпјҢдҪ еңЁеҶӣи®ӯз»“жқҹж—¶дә§з”ҹиҝҷж ·зҡ„з–‘й—®пјҢе№¶дёҚжҳҜе·§еҗҲгҖӮ

зҺ°е®һз”ҹжҙ»дёӯеӨ§и§„жЁЎеӘҡдҝ—зҡ„еңәйқўиҝҳжңүеҫҲеӨҡпјҢжҜ”еҰӮеҚҮж——д»ӘејҸпјҢйҳ…е…өејҸгҖҒе©ҡзӨјпјҢжғ…дәәиҠӮзҡ„зҺ«з‘°гҖҒжҜҚдәІиҠӮзҡ„еә·д№ғйҰЁпјҢй’“йұјеІӣдәӢд»¶еҗҺдёҠиЎ—зҡ„зҲұеӣҪжёёиЎҢпјҢиЎЎж°ҙдёӯеӯҰжҒҗжҖ–зҡ„й«ҳиҖғиӘ“еёҲеӨ§дјҡзӯүзӯүгҖӮ

зұіе…°жҳҶеҫ·жӢүе№¶дёҚжҳҜиҰҒеҸҚеҜ№ж„ҹжғ…зҡ„жөҒйңІпјҢжӣҙдёҚжҳҜеҸҚеҜ№ж„ҹжғ…жң¬иә«гҖӮд»–еҸҚеҜ№зҡ„жҳҜйӮЈз§ҚеңЁеӨ§дј—дёӯй«ҳеәҰз¬ҰеҸ·еҢ–зҡ„жғ…ж„ҹеҸҚеә”пјҢеӣ дёәиҝҷйҒ®и”Ҫз”ҡиҮіжүӯжӣІдәҶдәәзҡ„зңҹжӯЈжғ…ж„ҹпјҢз”ҡиҮіеҪўжҲҗдәҶдёҖз§Қжғ…ж„ҹжҡҙеҠӣпјҢеҜ№дёӘдәәиҝӣиЎҢз»‘жһ¶гҖҒеҲ©з”ЁгҖӮжҜҚдәІиҠӮзҡ„еҲӣе§Ӣдәә е®үеЁң.иҙҫз»ҙж–Ҝзҡ„еҗҺеҚҠз”ҹйғҪеңЁе‘јеҗҒеҸ–зј”жҜҚдәІиҠӮпјҢеӣ дёәеҘ№еҸ‘зҺ°пјҢжҜҚдәІиҠӮе·Із»Ҹе®Ңе…Ёиў«е•ҶдёҡеҢ–дәҶпјҢеҫҲеӨҡдәәйқ еҚ–еә·д№ғйҰЁеҸ‘дәҶеӨ§иҙўгҖӮеҸҰеӨ–пјҢзҹҘд№ҺдёҠзҡ„з”·еҗҢиғһпјҢжңүжІЎжңүдёҖеҲ°жғ…дәәиҠӮе°ұжҠ“зӢӮзҡ„ж„ҹеҸ—пјҹеҫҲеӨҡдәәдёҖе®ҡжңүеҗ§гҖӮ

зӢ¬з«Ӣе’ҢиҮӘз”ұжҳҜеұһдәҺеӢҮж•ўиҖ…зҡ„гҖӮе”ҜжңүйӮЈдәӣж•ўдәҺжҠҠиҮӘе·ұи§Јеү–зҡ„иЎҖж·Ӣж·Ӣзҡ„з»ҷиҮӘе·ұзңӢзҡ„дәәпјҢжүҚиғҪиҺ·еҫ—зӢ¬з«Ӣе’ҢиҮӘз”ұгҖӮйӮЈдәӣе»үд»·зҡ„зңјжіӘпјҢжҳ“еҫ—зҡ„ж„ҹеҠЁпјҢеҸӘжҳҜејұиҖ…иҜҒжҳҺиҮӘе·ұжҙ»иҝҮзҡ„е®үж…°еүӮгҖӮ

еҲ°еә•ж„Ҹйҡҫе№ігҖӮ

жӣҫз»Ҹеӣ дёәеҜ»еҰҲи®°еәҰиҝҮиҮӘе·ұзҡ„еӨұжҒӢжңҹпјҢеҫҲе°‘еҜ№и°ҲзҲұжғ…зҡ„еү§и®ӨзңҹпјҢжүҖд»ҘиҝҷдёӘз»“еұҖпјҢеҲ°еә•ж„Ҹйҡҫе№ігҖӮ

еҰӮжһңиҝҷдёҚжҳҜдёҖйғЁжғ…жҷҜе–ңеү§пјҢиҖҢжҳҜдёҖйғЁ40-60еҲҶй’ҹзҡ„йғҪеёӮзҲұжғ…зҺ°д»Јеү§пјҢжҲ‘дёҚдјҡжңүиҝҷд№ҲеӨ§зҡ„жҠөи§Ұжғ…з»ӘпјҢжғ…жҷҜе–ңеү§д№ӢжүҖд»Ҙдёәжғ…жҷҜе–ңеү§пјҢеңЁдәҺдёҚз®ЎжҗәеёҰеӨҡе°‘йҡҗе°„зҺ°е®һзҡ„з§Ғиҙ§пјҢз»“еұҖеҝ…е®ҡиҜҘжҳҜжІ»ж„Ҳзҡ„пјҢжӯЈиғҪйҮҸзҡ„пјҢе……ж»ЎеёҢжңӣзҡ„пјҢдјҡи®©дәәи§үеҫ—еҸҜз¬‘еҸҲзҫЎж…•еҗ‘еҫҖзҡ„пјҢз”ҡиҮіеҸҜд»ҘжңүзӮ№з«ҘиҜқе®үжҠҡж„Ҹе‘ізҡ„Happy EndгҖӮ

дҪҶеҫҲжҠұжӯүпјҢжҲ‘жІЎи§үеҫ—HIMYMзҡ„з»“еұҖжңүиҝҷж ·зҡ„еҖҫеҗ‘пјҢж— е…іжҲ‘жҳҜBRе…ҡиҝҳжҳҜTRе…ҡпјҢжҲ‘еҸӘжҳҜи§үеҫ—иҝҷдёӘз»“еұҖд»ҺеӨҙеҲ°е°ҫжү“дәҶеүҚйқўе…«еӯЈ+S9E1-22й“әеһ«зҡ„и„ёгҖӮ

жү“дәҶиҖҒеҰҲзҡ„и„ёпјҢжү“дәҶTedзҡ„и„ёпјҢжү“дәҶе·ҙе°јзҪ—е®ҫзҡ„и„ёпјҢжү“дәҶжҲ‘д»¬иҝҷзҫӨйҷӘзқҖзңӢпјҢи·ҹзқҖе“ӯпјҢйҖ—еҲ°з¬‘зҡ„еҝ е®һи§Ӯдј—зҡ„и„ёгҖӮ

жү“дәҶжүҖи°“зҡ„TMDзҡ„зңҹзҲұзҡ„и„ёгҖӮ

иҝҷе“ӘйҮҢжҳҜжғ…иҠӮе–ңеү§пјҹ

иҝҷеҲҶжҳҺжҳҜдёҖйғЁи®Ҙи®ҪзҲұжғ…и„ҶејұпјҢеҳІз¬‘зңҹзҲұе…ҡпјҢйј“еҠұзүәзүІдёҺеҰҘеҚҸзҡ„зҺ°е®һеү§гҖӮзј–еү§дҪ жҳҜиў«дәәз”©дәҶжҠҘеӨҚзӨҫдјҡеҗ—пјҹжҳҜеҳӣпјҹдҪ жҳҜеңЁйҖ—жҲ‘еҗ—пјҹ гҖҖгҖҖ

е·ҙе°је’ҢзҪ—е®ҫзҰ»е©ҡжҲ‘еҸҜд»ҘеӢүејәжҺҘеҸ—пјҢеҶҚеӨҡжҝҖжғ…жүҝиҜәзҡ„зЎ®жһ¶дёҚдҪҸзҺ°е®һжӢүй”ҜпјҢзҪ—е®ҫе’Ңе·ҙе°јд№ӢжүҖд»ҘеҲҶеҲҶеҗҲеҗҲд№ҹжҳҜд»–д»¬дёӨдёӘжҖ§ж јжүҖиҮҙпјҢдёӨдәәйғҪжҳҜжҖ§ж јејәпјҢејәи°ғиҮӘжҲ‘пјҢдё”еҜ№иҮӘиә«еӨ–зҡ„иҙЈд»»зјәд№ҸжҢҒд№…жӢ…еҪ“зҡ„дәәпјҢиҝҷз§ҚдәәеҸҜд»ҘдёәиҮӘе·ұзҡ„жңӢеҸӢдёӨиӮӢжҸ’еҲҖпјҢеҸҜд»ҘеңЁиҮӘе·ұзҡ„дәӢдёҡдёҠжүҖеҗ‘жҠ«йқЎпјҢеҚҙзӢ¬зӢ¬е®№жҳ“й”ҷиҝҮзҲұжғ…гҖӮзҲұжғ…зҡ„жң¬иҙЁе…¶е®һжҳҜдёӨдёӘдәәеңЁзЈЁеҗҲдёӯж„ҝж„ҸзүәзүІдёҖйғЁеҲҶзҡ„иҮӘжҲ‘жқҘдәӨжҚўеҜ№ж–№зҡ„зҲұдёҺеқҡе®ҲгҖӮжӯЈеҰӮиҺүиҺүе’Ңй©¬дҝ®жүҖз»ҸеҺҶзҡ„йӮЈж ·пјҢиҺүиҺүе’Ңй©¬дҝ®йғҪдёәдәҶе©ҡе§»е’Ңе®¶еәӯж”ҫејғдәҶдёҖйғЁеҲҶиҮӘе·ұзҡ„зҗҶжғіпјҢдҪҶд»–д»¬зҡ„зҲұжғ…еҺҶд№…ејҘж–°ең°д№…еӨ©й•ҝгҖӮ

е·ҙе°је’ҢзҪ—е®ҫиӮҜе®ҡжҳҜзңҹзҲұиҝҮеҜ№ж–№зҡ„пјҢдҪҶиҝҷз§ҚзңҹзҲұжҳҜе»әз«ӢеңЁжҖ§ж јзҡ„зӣёеғҸдёҠпјҢжҳҜе…ұжҖ§зӣёеҗёпјҢжүҖд»Ҙд»–д»¬еҸҜд»ҘжңүзҲұжғ…пјҢдҪҶжІЎжі•з»ҙжҢҒе©ҡе§»пјҢBRе…¶е®һеңЁз»“е©ҡзҡ„ж—¶еҖҷйғҪеҒҡеҮәдәҶдёҖе®ҡзҡ„и®©жӯҘпјҢдҪҶжҳҜе©ҡе§»жҳҜдёӘжҢҒз»ӯзүәзүІзҡ„иҝҮзЁӢпјҢиҖҢд»–д»¬ж”Ҝд»ҳдёҚдәҶпјҢжүҖд»ҘжңҖеҗҺе’Ңе№іеҲҶжүӢпјҢеӣһеҪ’жң¬жҖ§гҖӮ

жңҖз»ҲпјҢзҪ—е®ҫиҺ·еҫ—дәӢдёҡдёҠзҡ„жҲҗеҠҹпјҢе·ҙе°јеӣ дёәдёҖж¬Ўж„ҸеӨ–иҺ·еҫ—дәҶеҸҰдёҖз§Қе®¶еәӯзҡ„еҪ’еұһпјҢзңӢеҲ°иҝҷйҮҢпјҢжҲ‘иҷҪз„¶жңүзӮ№е өпјҢеҘҪжӯ№иҝҳиғҪжҺҘеҸ—гҖӮе°ұеҪ“з¬¬д№қеӯЈжүҖжңүе©ҡзӨјиҝҮзЁӢиў«зӢ—еҗғдәҶеҗ§гҖӮе“ҰпјҢжҳҜиў«зј–еү§еҗғдәҶеҗ§гҖӮ

дҪҶжҳҜз»“еұҖжі°еҫ·еұ…з„¶еҸҲеҺ»дёҫзқҖи“қиүІе°ҸеҸ·зәҰзҪ—е®ҫдәҶпјҢжҲ‘з®ҖзӣҙеҚ§дәҶдёӘеӨ§ж§ҪпјҒпјҒпјҒпјҒ

жі°еҫ·жң¬иә«д»ЈиЎЁзқҖзҡ„жҳҜдёҖз§ҚеҜ№зңҹзҲұзҡ„еӢҮж•ўжү§зқҖиҝҪжұӮпјҢжҲ‘жҗһдёҚжҮӮзј–еү§еңЁжү“дәҶвҖңжөӘеӯҗеҘіејәдәәжҗһдёҚжқҘе©ҡе§»вҖқиҝҷж ·зҡ„зүҢеҗҺпјҢеұ…з„¶иҝҳж•ўжү“вҖңиҝҪжұӮзңҹзҲұзҡ„жң¬иҙЁе°ұжҳҜдёӘzuoвҖқиҝҷж ·зҡ„зүҢпјҒпјҒ

жі°еҫ·д№ӢжүҖд»ҘдјҡдёҖж¬Ўж¬ЎжҒӢзҲұеӨұиҙҘпјҢжҳҜеӣ дёәд»–е§Ӣз»Ҳи§үеҫ—еҜ№ж–№дёҚжҳҜеҘ№зҡ„зңҹе‘ҪеӨ©еҘіпјҢиҖҢжңҖеҗҺиҖҒеҰҲеҮәжқҘд№ҹе°ұйў„зӨәзқҖиҰҒиҜҒжҳҺе…Ёеү§зҡ„з»ҲжһҒе‘ҪйўҳвҖ”вҖ”зңҹзҲұж— д»·пјҢеқҡе®ҲеҸҜиҙөгҖӮе®Ҳеҫ—дә‘ејҖи§ҒжңҲжҳҺпјҢжңқжңқжҡ®жҡ®еҸӘдёәMiss.RightгҖӮ

иҖҒеҰҲзҡ„е°Ҹй»„дјһпјҢеҗҚеӯ—пјҢжҖ§ж јпјҢе©ҡзӨјдёҠзҡ„еҜ№зңјпјҢйғҪжҳҜе‘Ҫйўҳзҡ„иЎҘе……иҜҒжҳҺпјҢжңҖеҗҺе°ұз®—иҖҒеҰҲжӯ»дәҶпјҢиҖҒзҲёеҸЁеҸЁеҸЁеҸЁзҡ„иҝҪеҝҶеҫҖжҳ”пјҢд№ҹеҸҜз®—жҳҜдёӘзҫҺж»Ўзҡ„ж•…дәӢпјҢз»“жһңдёӨдёӘзҶҠеӯ©еӯҗдёҖи„ёпјҡвҖңзҲ№дҪ зҡ„зңҹзҲұжҳҜзҪ—е®ҫйҳҝе§Ёе•ҰпјҢжҲ‘еҰҲеҰҲжҳҜи°Ғе•ҰжҲ‘д»¬иҮӘе·ұйғҪдёҚеңЁд№ҺпјҢж”ҫжүӢеҺ»иҝҪйҖҗзңҹзҲұеҗ§пјҢд№Ҳд№Ҳе“’пјҢзҲёжҜ”гҖӮвҖқ

еҚ§вҖ”вҖ”ж§ҪвҖ”вҖ”пјҒпјҒпјҒпјҒ

иҖҒеҰҲе“ӘйҮҢз®—й…ұжІ№пјҢиҖҒеҰҲе°ұжҳҜдёӘзғҹйӣҫеј№пјҢйўҳзӣ®дёҠеҝҪжӮ иҝҳдёҚеӨҹпјҢеҹӢдәҶйӮЈд№ҲзәҝпјҢиҝҳдё“й—ЁжүҫдәҶжј”е‘ҳжқҘжј”й—ӘеӣһпјҢе…¶е®һеҸӘжҳҜдёәдәҶеҝҪжӮ и§Ӯдј—иҖҢе·ІпјҢзј–еү§дҪ иҝҷжҳҜеңЁиЈ…йҖјдҪ зҹҘйҒ“еҗ—пјҒпјҒпјҒпјҒпјҒ

иҖҒе®һиҜҙпјҢзңӢеҲ°жңҖеҗҺз»“еұҖпјҢжҲ‘зңҹеҝғдёҚи§үеҫ—жі°еҫ·е’ҢзҪ—е®ҫеңЁдёҖиө·жҳҜд»Җд№ҲвҖңи“Ұз„¶еӣһйҰ–йӮЈдәәеҚҙеңЁзҒҜзҒ«йҳ‘зҸҠеӨ„вҖқзҡ„зңҹзҲұи§үйҶ’пјҢж №жң¬е°ұжҳҜдёӨдёӘеӯӨе®¶еҜЎдәәжңҖеҗҺеҶіе®ҡеҮ‘еҗҲзқҖиҝҮз®—дәҶпјҢиҝҷдёҚе°ұжҳҜиҖҒжқҘдјҙеҳӣпјҒпјҒпјҒжі°еҫ·иҮӘе·ұд№ҹиҜҙд»–е·Із»ҸдёҚжҳҜзҲұзқҖзҪ—е®ҫзҡ„йӮЈдёӘдәәдәҶпјҢзҪ—е®ҫд№ҹеҜ№жі°еҫ·иҜҙиҝҮиҰҒlet it goпјҢеҘ№еҜ№жі°еҫ·жӣҙеӨҡзҡ„жҳҜеҜ№е®¶дәәзҡ„дҫқиө–е’Ңд№ жғҜгҖӮ

иҝҷд№ҲжҳҺжҳҫзҡ„вҖңеҮ‘еҗҲиҝҮвҖқдҪ еңЁжңҖеҗҺиҝҳзӮ№д»Җд№ҲзӮ№пјҢиҰҒжҗһеҫ—иҝҷд№ҲзҺ°е®һеҗ—пјҒпјҒпјҒпјҒ

жүҖд»ҘеӣһеҲ°жңҖеҲқзҡ„жңҖеҲқпјҢжҲ‘дёҚжҳҜиҜҙзҗҶи§ЈдёҚдәҶз»“еұҖпјҢжҲ‘жҳҜжҺҘеҸ—дёҚдәҶиҝҷж ·зҡ„з»“еұҖдҪңдёәжғ…жҷҜе–ңеү§HIMYMзҡ„з»“еұҖпјҢд»Җд№ҲзҺ©ж„Ҹе„ҝпјҹ

дҪ иҰҒжҳҜжңҖеҗҺеҲ«и®©иҖҒеҰҲеҮәжқҘпјҢжі°еҫ·еҗҺжқҘйҡҸдҫҝжүҫдёӘеҰ№еӯҗе©ҡдәҶпјҢеӣһеӨҙжӯ»дәҶеҶҚжүҫзҪ—е®ҫжҲ‘д№ҹиғҪжҺҘеҸ—гҖӮйӮЈеҖ’зңҹзҡ„з®—жҳҜд»ҺдёҖиҖҢз»ҲзңҹзҲұе”ҜдёҖпјҢд№ҹз®—з¬ҰеҗҲжі°еҫ·зҡ„дәәи®ҫгҖӮзҺ°еңЁж•ҙзҡ„дёҖеүҜвҖңдёҖз”ҹдёҖдё–еҜ»еҘ№еҚғзҷҫеәҰз»ҲдәҺеҺ®е®ҲдёҖз”ҹе„ҝеҘіиҶқдёӢвҖқзҡ„жһ¶еҠҝпјҢз»“жһңжӯ»дәҶпјҢе°ұеҸ‘зҺ°зңҹзҲұжҳҜзҪ—е®ҫпјҢзј–еү§дҪ ж·ұдә•еҶ°е•ҠпјҒпјҒ

еңЁзҪ‘дёҠжҗңдәҶдёҖеӨ§е Ҷиө„ж–ҷд»ҘеҗҺеҺҹжқҘжҳҜеӣ дёәDjangoзҡ„Cross Site Request Forgery protectionжңәеҲ¶зҡ„й—®йўҳгҖӮиҝҷдёӘжңәеҲ¶жҳҜдёәдәҶдҝқжҠӨдёҚеҸ—csrfж”»еҮ»гҖӮи§ЈеҶіж–№жЎҲDjangoзҡ„е®ҳзҪ‘е·Із»ҸжҸҗдҫӣдәҶhttp://docs.djangoproject.com/en/dev/ref/contrib/csrf/пјҢж №жҚ®иҜҙжҳҺдҝ®ж”№еҗҺпјҢajaxеҸҜд»ҘйЎәеҲ©PostдәҶгҖӮ

е…·дҪ“еҠһжі•жҳҜпјҢйҰ–е…Ҳи§ЈеҶіиЎЁеҚ•зҡ„POSTгҖӮеңЁsettings.pyж–Үд»¶дёӯжүҫеҲ° MIDDLEWARE_CLASSESпјҢеңЁе…¶дёӯеҠ е…ҘдёҖдёӘдёӯй—ҙд»¶пјҡвҖҷdjango.middleware.csrf.CsrfViewMiddlewareвҖҷ пјҢдҝ®ж”№еҗҺзҡ„д»Јз ҒеҰӮдёӢпјҡ

з»ҸиҝҮиҝҷз•Әдҝ®ж”№пјҢеҸҜд»Ҙи§ЈеҶіиЎЁеҚ•ж–№ејҸзҡ„POSTжҸҗдәӨHTTP 403й—®йўҳдәҶгҖӮajaxзҡ„PostжҸҗдәӨд»…иҝҷж ·ж”№иҝҳдёҚиЎҢгҖӮиҝҳйңҖиҰҒй’©дёҠдёҖдёӘжҜҸж¬ЎжҸҗдәӨж—¶зҡ„cookieеӨ„зҗҶиҝҮзЁӢгҖӮд№ҹе°ұжҳҜжҜҸж¬ЎжҸҗдәӨзҡ„ж—¶еҖҷпјҢйғҪи§ҰеҸ‘иҝҷдёӘиҝҮзЁӢпјҢеңЁжҸҗдәӨзҡ„httpеӨҙдёҠеҠ е…Ҙcsrf tokenгҖӮдёҚиҝҮиҝҳеҘҪпјҢеҰӮжһңдҪ жҳҜз”ЁjQueryжқҘеӨ„зҗҶajaxзҡ„иҜқпјҢDjangoзӣҙжҺҘйҖҒдәҶдёҖж®өи§ЈеҶій—®йўҳзҡ„д»Јз ҒгҖӮжҠҠе®ғж”ҫеңЁдёҖдёӘзӢ¬з«Ӣзҡ„jsж–Үд»¶дёӯпјҢеңЁhtmlйЎөйқўдёӯйғҪеј•е…ҘеҚіеҸҜгҖӮжіЁж„ҸиҝҷдёӘjsж–Үд»¶еҝ…йЎ»еңЁjqueryзҡ„jsж–Үд»¶еј•е…Ҙд№ӢеҗҺпјҢеҶҚеј•е…ҘгҖӮд»Јз ҒжҲ‘зӣҙжҺҘеӨҚеҲ¶иҝҮжқҘдәҶпјҢеҰӮдёӢпјҡ

$('html').ajaxSend(function(event, xhr, settings) {
    function getCookie(name) {
        var cookieValue = null;
        if (document.cookie && document.cookie != '') {
            var cookies = document.cookie.split(';');
            for (var i = 0; i < cookies.length; i++) {
                var cookie = jQuery.trim(cookies[i]);
                // Does this cookie string begin with the name we want?
                if (cookie.substring(0, name.length + 1) == (name + '=')) {
                    cookieValue = decodeURIComponent(cookie.substring(name.length + 1));
                    break;
                }
            }
        }
        return cookieValue;
    }
    if (!(/^http:.*/.test(settings.url) || /^https:.*/.test(settings.url))) {
        // Only send the token to relative URLs i.e. locally.
        xhr.setRequestHeader("X-CSRFToken", getCookie('csrftoken'));
    }
});
