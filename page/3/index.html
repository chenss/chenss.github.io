原来我不是最努力的

五天的安全培训结束了，在兖州的集训也要开始了。在聊城的那几天，大多数人跟我一样，没认真听多少东西。但考试时才发觉还是有差距的。虽然已经给了答案，已经背得差不多了，真写起来还是会忽然记不清。至少在应试上，我连临阵磨枪的耐心都没有。

回家时还特意把教材带回去。的确也看了一些，相较于上学时，进步了。回来了才发现，原来不只我知道预习。是的，他们没回家，在这太无聊了，学了，比我学得多点也可以理解。

我应该更努力啊！与周围的人相比，至少从大学开始，我已经不是比较聪明的那类人了。如果还像过去那样，注定又要倒数了。

我不是自责，真的要努力，坚持努力。如果他们也一直在努力，那你就要更努力。如果他们在家便放松了，而你还在忙碌，你就赚到了！

可怕的工作

我像期待中学与大学一样，期待着工作与工资，满怀热情。但拖来拖去，直到今天还没开始，恐慌开始在我的心中蔓延。今天闫鸿儒竟破天荒地主动与我聊天，聊她心里的压力，聊她想辞职。这不是给我添乱吗!

我原本对未来“充满”了规划，还想着一有机会就考工程师呢。。。她给我的感觉却是，我更应该考虑是否要辞职。我知道你压力大，但我真没打算辞职，至少近几年不会的。这些天那么闲，我可是东一下，西一下地想了好多呢。辞职，那这些，就白想了。想想就不甘心啊。当然，主要还是工作还不错，要不我还憧憬个屁啊。

比工作更让人兴奋的，当然是工资。这是我最重要的不考研的理由。我要独立，首先要经济独立。我要用自己的钱给自己买东西，我要送别人自己的东西。

又跑题了，我应该聊聊工作有多可怕。它可怕到让人失去激情（我对任何事情都极有可能迅速失去激情）。它还让人反复无常。某人不就是如此吗，发的说说那么反复。今天感觉前途渺茫，明天就非常兴奋。或者一会儿抱怨太累，要请假，到点还是乖乖上班去了。又是扣工资，扣资金，又是能吃到大包子真好。心情起起伏伏，像股票一样无法预测。我也会如此吧，只是很快，这一切都会变成无所谓，变得冷漠，虽不会因抱怨而话多，却因沉默而话少。

所以，规划。

Dos批量重命名

注：只适用于在文件尾部添加字符的情况，两条命令搞定。

例如我要批量重命名E盘下的几本Mobi文件，进入相应目录后命令如下：

rename *.mobi *.

此时会批量去掉扩展名（rename时不加“.”会失败，但即使加了也不会有“.”，不知道为什么）。

然后：

rename * *_test.mobi

此时的文件名与最初相比，只是在文件名后，扩展名前加了“_test”

最后一批离开的人

我是第一批领毕业证的人，当晚便回家了。其实距谁今也只有十几天的时间，却感觉那么遥远。

先说一下这几天的行程吧：

  • 6月29号的火车，30号早上才到家，歇了一天。
  • 7月1号用一天时间收拾了房间。
  • 2号不知道具体干了什么，不过这天应该也在计划找工作的事。
  • 3号去了济南，早上不到5点便起床去赶火车。
  • 4号又去了济南，由于意外原因，不得不立刻回到安大，没能回家。
  • 5号在吴若英那睡了午觉。几日来心神疲惫，这是第一次好好地睡觉。午饭也是几日来吃得最正常的一次。下午吴若英、张骁勇和我一起去吃饭。当夜赶火车回济南。
  • 6号下午终于到了济南。好好地洗了澡，洗了袜子（事出匆忙，未带任何换洗的衣物）。
  • 7号又回了安大，晚上才到。
  • 8号洗了衣服，无法出门，歇了一天。
  • 9号改签、收快递，张骁勇请吃饭。
  • 10号清楼，从宿舍搬了出来，白天在档案学实验室，晚上在吴若英处睡。
  • 11号去庐江，吃饭、钓鱼。
  • 12号下雨，哪也没去成，下午回了合肥。然后坐了一夜的火车，各种周折之后
  • 13号上午到家。

我真的想你们了。

第一个离开的人

没想到，也没想，成为第一个离开的人。跑来跑去，终于忙完所有事情，拿到了毕业证，这就是忙碌的今天。

我不得不尽快回去。也不必须今天，明天可以，后天也可以，只要尽快就可以。我习惯性地选择忽视这种感受，而选择更实际的事。我不信任自己的感觉，面对外界冲突的时候，大多数情况下。

我向来希望任务结束后的情感放纵。期末结束了便不必担心考试，备考的压力也让我更积极的宣泄情感。越重大，越艰难的任务，结束的那一刻我的心越脆弱，越不设防。

期末结束了，我们欢乐的回家了；高考结束了，我们欢呼着回家了。论文答辩结束了，我们有毕业晚宴。我无数次的羡慕着美国的中学生。他们不是一考定终身，临近毕业可以有很多集体的活动。虽然思维还清晰，控制力却差了，毕业视频里的我走路都摇晃了。

感谢徐凯、方鼎、刘献栋为我送行，这是第一次，第一次回家的时候有人专门送我。姐姐问我为什么看电视剧的时候从来不哭。无论情节如何感人，凄凉，悲苦。这是事实，我只会被自己感动。事前预想时，事后回忆起，独自一人时，我更可能被自己的所思所想感动。徐凯，毕业晚宴上，你抱着我哭的时候，我挺高兴的。我们的四年没有白过，这些眼泪就是收获。谢谢你们陪我。

路过市口腔医院，我想起三次来到这里。路过安医大，对不起，我首先想起了另一个人，然后意识到昨天和你们，在这里看电影。路过安大老区，我想到了一个室友，一个学姐。路过XX大酒店，我想到了毕业晚宴。路过稻香村，我想到了小潘，小夏，小丸子（别乱想，没有俞老师）。

我会热爱新的生活，我会珍惜新的朋友，但不会忘记你们。我会想你们的。

贫穷或富有

如有那份觉悟或妥协，贫穷或富有也不见得那么重要。但事事皆有度，离开剂量谈毒性都是耍流氓。富有对人有坏的影响，可只要是凭自己的力量而富有，大多人应该也应付得来。贫穷的坏处，就难以避免了。贫穷的人远远多于富有，这其中既有看淡财富和时运不济的，更大大地有自身能力不足的。

贫穷会束缚人的见识，身体不得自由的同时思想也不得自由。我不强求大富大贵，但也不愿贫困。我不求超凡脱俗，也不能俗不可耐。

我不希望成为自己讨厌的人。

hexo速度优化(landscape主题)

Hexo原本默认的主题是light，很多解决方法都是针对这个主题的。后来Hexo默认的主题改为了landscape，我用的就是这个。这几天天朝对google的封锁尤为厉害，我不得不优化一下速度了。

首先是fonts.googleapis.com，也就是google字库载入速度太太太慢的问题，只需把 themes/landscape/layout/_partial/head.ejs 里的 <link href="http://fonts.googleapis.com/css?family=Source+Code+Pro" rel="stylesheet" type="text/css"> 注释掉或者直接删掉就行了。

jquery.min.js 载入过慢的解决方法见此：http://charyun.com/2014/06/08/hexo-blog-access-speed/其实就是把google的CDN换成了sina的。实在无聊的话也可以试试这些：国内有哪些靠谱的 Javascript 库 CDN可用？

未经详细测试，欢迎反馈。
