MacBook Air

至今为止，能激起我强烈赚钱欲望的，除了女人，也只有电脑了。

带着厚厚的三星笔记本，还得带着电源线、鼠标，一点都不潇洒。Windows逼格太低，Linux太麻烦。Mac OS多好，桌面环境稳定漂亮，命令行也丝毫不差。反正我既喜欢MacBook Air的硬件，也喜欢软件。只是好贵。

我看了一遍又不遍MacBook Air的官网，想着教育优惠，等着圣诞促销，算着分期付款。甚至会告诉自己在赚钱以前不能再看这些。就好像想要六维邀请码，拼命赚百度积分的日子。我浪费了太多时间在这些细枝末节上，却没有为了买mba而去兼职赚钱。

我也想过装黑苹果，至少先体验一下苹果的系统。对于从什么都不懂，独自一人装过Windows、装过各种Linux，包括Arch linux的我来说，这应该不是无法完成的任务。大不了多折腾折腾，以往的哪个系统不是折腾了两三天才装好的。可黑苹果，我终于失败了。大四了，烦人的事太多，而且没那么乐意折腾了。后来虽然还想过，但想想又忍住了——直接买mba不行吗！

算来算去，工作第一年我是一点钱都不打算存了。

每天进步一点点

在中国目前的情况下，为什么要买正版？

来源：http://www.zhihu.com/question/23735611

本人研究生转到计算机，虽然现在从事的是这个行业，但是依然不明白为什么要支持正版。

1）有人说用盗版可能会有木马，不安全——我不在乎；

2）还有人说：如果软件公司挣不到钱纷纷倒闭，我们就不可能有好软件用了——事实是我们都知道不可能所有软件公司都倒闭；

3）还有一个说法，用音乐行业来类比：如果大家都下载盗版音乐，那音乐人挣不到钱，就不会有好音乐了。自私点说，音乐人做出音乐就好，更何况他们也不可能挣不到钱；就算退一步说，他们挣不到钱了，跟听的人有什么关系，我们只要能下载听就好。这个音乐人如果因为挣不到钱不做音乐了，我们还可以去下载其他音乐人的作品听，就像 2）中所说，不可能所有的音乐人都消失的，那为什么要买正版音乐支持他们。

所以为什么就一定要支持几百的正版而不是下载盗版？

因为我有钱，而且懒得折腾。

楼上大多数的回答，都没有答到点子上。
其实提问者这样的人，在中国是主流，他们在大街小巷、在城市乡间、在各个论坛、也在我们每个人身上。
而法律、道德，在离主流的中国人很远，偶尔有那么几次正版风暴，也无非是雾里看花。所以，版权保护这个观念，不管是从法律角度还是道德角度来说，对中国社会都很超前。
——你对着一个300年前的普通中国人讲女权讲人人平等，还希望他也能理解这是天经地义的事情，你觉得会有效果吗？根本不在一个频道啊！

所以，让我们务实一点，说说为什么要用正版。

1.我年纪大了，比过去有钱，比过去没时间。
小时候省下零花钱也只能买得起69块钱的国产游戏，愿意花几个小时去论坛里找破解补丁（有时候也只有几分钟哈）、感谢大神们的0day发布，每次升级后重新破解……我那时候真不觉得累，因为我的时间不值钱。
小时候我还从VeryCD拖很多MP3下来，手动更新里面的IDv3信息，统一码率，文件摆放得井井有条，那都算一种乐趣。
小时候找来txt格式的电子书，自己用排版软件重新格式化……当然后来是有少数一些制作比较精良的pdf和epub了。
现在虽然不是富豪，但算是衣食无忧，有点闲钱。正版对我来说，代表立等可取，代表不用折腾，代表可以获得开发者的支持，代表我可以省下更多时间去享受。

2.正版比以前更便宜了，也更方便了，质量更高了。
曾经有个时代，盗版反而比正版更方便。从务实的角度来说，除了人格高尚，我想不出有什么理由支持那种“偷懒”的正版开发者。
比如以前的正版游戏，引进中国后改头换面连升级补丁都不通用，还加了奇怪的DRM，反而是盗版直接硬盘版都不用怕伤光盘；
比如以前的正版音乐，非要买一个CD盒子回去，完全不考虑其实大家都不用CD机了，而且盒子争先恐后做得越来越大其丑无比；
比如以前的正版电影，影院里与世界不同步，正版DVD还制作粗糙缺斤少两。
但现在这个时代，聪明人会用正版。
现在的正版游戏，越来越多有数字发行，方便得要死，还能联机；（我是Steam用户）
现在的正版音乐，我可以用QQ绿钻去下载，非常便宜，而且IDv3的信息很齐全很规范；（补充：评论里有朋友说QQ绿钻的版权有问题）
现在的正版电影，在影院这端与世界越来越接轨了（你看，这就是正版的好处，看电影的人多了，片方就会重视这个市场，就像港台总能有官方中文版游戏一样），在家庭播放这一端也越来越多的正版授权了（广告或会员订阅）。
现在的正版电子书，有封面有注释有笔记，字体适当排版漂亮，而且还会经常修订更新。
而且正版之后会越来越好，越来越丰富和方便。

3.正版的定义正在变化。
以前大家理解的正版，是花人民币去买了那个盒子回来。盒子和光盘（实体）丢了，谁也不能证明你拥有正版。
现在的正版不一样了，数字化之后，你是不是拥有正版，很容易证明。
我丢了实体根本不要紧，东西都在云端，我随时可以找回来，不用再次付费。
我的记录也都在云端，我的文档、状态、游戏存档、电影断点都在云端，换个设备就能继续，请问哪个盗版商可以提供这种服务？
我的阅读进度也在云端，而且电子书如果更新了也会同步推送。盗版电子书制作精良的我见过，同步更新的真没见过，为什么？因为那只是一个孤立的文件，而不是服务。
甚至，有时候我用正版，根本不用花钱。因为我付出了我的个人信息（包括部分隐私），接受一些广告，我就能使用正版。所以正版不一定等于花钱，而是有付出。
这个时代，不理解数字发行的变化，你就不能理解为什么要用正版。

4.选择面越来越广
以前的时代是很畸形的，因为收费软件都是“免费”的，所以大家也就很依赖这些软件。
现在对我来说，正版化未必就是花钱买收费软件。在互联网如此发达、版权授权又更灵活的时代，其实有很多不同的选择，让普通个人更有办法做到正版化。
例如，MS Office之类的天价（其实个人版也不贵了哈哈）软件，我如果不工作的话完全是用不上的（个人使用的话杀鸡真不用牛刀），如果工作的话它的确是非常强大难以撼动，所以我的公司会帮我买下企业授权。Photoshop同理。这是一个挺好的判断在职企业是不是靠谱的方法呢。
例如，有更多免费的、开源或不开源的替代品，在非工作场合都足以完成任务。——以前人看图只知道ACDSee，哪听说那么多小众软件啊。
例如，音乐我越来越不用下载，而是在线收听，版权压力由豆瓣们虾米们去承担，发行商和这些网站们打交道，比和那么多普通用户打交道要省事多了。
例如，原来的命根子Windows，后来不仅因为降价而买了正版，再后台用了Mac后突然发现我不用操心操作系统了……
一个固步自封的人，才感觉不到选择面的拓广。正版化代表了我本人的进步，我并不希望到了30岁我还和20岁一样，天天去找各种破解资源，那说明我的虚拟生活半径没有任何变化。

自从我逐步正版化以来，我最大的感受就是“越来越自由”。所以正版不一定是种高高在上面貌严肃的束缚，不一定是道德的说教，而是一种实实在在的好处。相对这种好处，我觉得我的付出很便宜。

我满心期待未来的世界，正版会越来越不需要理由，因为它够方便、够便宜，以至于你忽略了正版的存在。

反过来，正版也不能认为有了天命护体，就不用做任何事情了。在一个完全竞争的市场里，即便对方是非法的，你也要做到最好才能够资格去殊死搏斗。正版只有提供差异化的价值，才会给人们消费的理由。

最后强调一下，我不是说“没钱有时间，就有道理用盗版”了。支持正版人人有责，我只是给出了一个主流中国人更能接受的理由而已。

每天进步一点点

练字

我是不是太喜欢碎碎念了？看着自己写的大纲，真这么写只会是一篇毫无意义的流水账啊。

说起练字，不是第一次有这种念头了。现在突然又要练也算是想了结过去的一个心愿，希望能够进步一点点。的确，和很多事情相比，这些微不足道的小事，即使做成了，也带来不了多少好处，人们还是倾向于干些实实在在的事。一切人们觉得有益，但每每又以很忙、太难为由，推脱不做的事，其实都不像他们所说的那样被认同。但假使真的做成了，即使是做为消遣，也是很值得高兴的。尤其是面对想做而没有做的人，总会有一种莫名的优越感。让别人明白什么叫“仕别三日”，连自己都刮目相看的时候，真的会很庆幸自己坚持下来了。至于实际的利益，总比做些无聊的消遣强吧。

但也别太折腾。每个人的时间和精力都是很有限的，最有效率的做法是同一时间只做一件事。以前对什么都感兴趣，但每个兴趣都很浅，什么都没学成也是情理之中的事。但也不是说兴趣多了就一定一事无成，关键是分得清轻重缓急并且有耐心。都说现在的人太浮躁了，总是急于求成。急于看到结果，于是加快了过程，结果必然不理想，没耐心的人便就此放弃了。这是最差的情况，付出却没收获。最好的情况是什么？没有最好，也没有更好。如果一开始就有耐心，在众多兴趣中专注于一件事，耐心的专注于过程，结果自然会早早的到来。然后就可以转战其它兴趣了。

即使结果没有早早地到来，也不用着急得六神无主。生活中的大多数事，本就是无法一次性做好的。妄想论文初稿就通过，只会陷入对细节无尽的苛求中。倒不如先糊弄个初稿出来。想糊弄初稿，唯一的办法就是有什么写什么，先写出来再说。练字不也是如此吗？只是它的阶段性太模糊，评价标准也模糊，相较之下，平添了些难度。

这仅仅针对不带功利的兴趣，否则我真的达不到这种境界。不带功利便不重结果，过程就是收获。有了功利，就需要有个结果；看重结果却不着急，这需要何等的自信啊！有耐心的自信，这才是真自信吧。即使阶段性成果不理想，也能冷静地判断，这才是真自信。

大纲：

  • 知乎：正确的握笔姿势、必须先练楷书吗、好用的钢笔、行书选谁的好、墨水
  • 练字的目的不强求写出好看的字，最重要的是能写得快。
  • 练字前的准备：尽量少写字、买字帖（五本 吴玉生）
  • 开始：五一集中写完五本字帖（实际未能按计划执行）
  • 初期以摹为主，准备了几张用过的打印纸，放在桌面上，随时练习
  • 打算抄一遍《万历十五年》，只抄了一章，不了了之。又改为另一本，抄了一页，不了了之。
  • 为写得快，最注重行楷中简化的写法，但看着别扭的不学。
  • 中期：仍以摹为主，遇到特别的随手在打印纸上练习。
  • 在字帖上临时，刚开始极关注在田字格中的相对位置，力求与字帖上相似。
  • 开始写第三本字帖了。
  • 后来写出了一些感觉，注重写每个笔划时的感觉。

http://zh.wikipedia.org/wiki/德沃夏克鍵盤

我是一个重度的Dvorak使用者，倘若给我一副普通的Qwerty键盘，我只能盯着它，用两个手指“戳字”。我的相册中有一张图片，是我正在使用的Macbook，按键已经被我重新排了座次。

Google了一通，似乎目前最快最流行的速记方法称为“亚伟速录”，使用专门定制的键盘和输入法。尽管无法在电脑上模拟，但是其核心内容大概有三点可以吸取：

  1. 根据中文习惯重新安排了键位，常用按键在中间一排。
  2. 将多个字符组成的韵母安排在了固定的按键上，类似“双拼”，只用输入一个字符。
  3. 如同使用乐器一般，多个按键可以同时按下。

Dvorak键盘布局下中文输入法的讨论几乎是个空白，我想在这一系列文章中对上面三点逐一分析，看看是否能够找到一些提高录入速度的办法。今天是第一部分，对Dvorak在中文输入的键位安排方面做一点研究。

import re  
file_name='new  4.txt'  
fp=open(file_name,'r')  
alllines=fp.readlines()  
fp.close()  
fp=open(file_name,'w')  
for eachline in alllines:

Hexo安装教程

参照这篇教程安装并优化了Hexo，但中间遇到了几个问题：

  1. 执行 npm install -g hexo 命令失败。

    主要是网络的原因（感谢伟大的GFW），可以多试几次，或者换个镜像。换镜像的方法：使用npm安装一些包失败了的看过来（npm国内镜像介绍）

    另外，Hexo官网给出的安装命令是 npm install hexo -g ，但似乎没影响。

  2. 添加多说评论/Jiathis分享时，找不到light目录。

    这个因为theme不同。Light主题是‘The default theme of legacy Hexo’，而Landscape是‘The default theme of Hexo 2.4+’。

    我的主题是Landscape，参考的使用hexo搭建静态博客：调整样式和使用Hexo搭建博客：使用Jiathis作为社会化分享工具

  3. 右键 Git Bash 的时候使用的是Windows自带的难用得要死的cmd.exe，怎么才能换成PowerCmd等第三方的软件？暂时只能在PowerCmd里cd到Hexo目录，然后进行操作。

一切完毕，上传到github之后就可以打开username.github.io看效果了。本地的整个hexo文件可以随意移动，复制到其它地方以后的操作和之前一样，只是目录换了而已。
